# Gifs App

Proyecto de practica enfocado en el desarrollo de componentes en React usando TypeScript.

---

## Tecnologías usadas
- React
- TypeScript
- Vite
- Vitest
- ESLint

---

## Configuración del proyecto
Este proyecto usa Vite como bundler y entorno de desarrollo.

### Ejecutar en local
```bash
npm install
npm run dev
```

### Ejecutar pruebas
```bash
npm run test
```
